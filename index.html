Welcome to blindwrite

blindwrite is a completly free website to learn 10 finger writing.
blindwrite is in a alpha state. Please report issues on github.

Important features:

  • completly free
  • no registration
  • supports special keyboard design
  • supports special and own keyboard layouts
  • no personal data stored on the server
  • open source
