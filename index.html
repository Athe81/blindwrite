Welcome to blindwrite

blindwrite is a completly free website to learn 10 finger writing.
blindwrite is in a early development state. Please report issues on github.

Important features:

  • completly free
  • no registration
  • open source
  • supports special keyboard design
  • supports special/own keyboard layouts
  • no personal data stored on the server
